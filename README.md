<h1>Super Contador</h1>
<p>Contador de passos digitavel simples</p>
<h2>Funciona assim: </h2> <br>
<ul>
  Primeiro: digite o passo inicial no campo início. <br>
  Segundo: digite o passo final no campo fim.<br>
  Terceiro: digite quantos passo darei sequencialmente. <br>
  Quarto: aperte o botão e o resultado virá instantaneamente.<br>
</ul>
